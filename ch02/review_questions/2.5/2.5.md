#### 解答

(a) 识别器(a|b|c)(a|b|c)(a|b|c)的DFA为

![](https://raw.githubusercontent.com/chenpengcong/EAC2-book-exercise-answers/master/ch02/review_questions/2.5/assets/2.5-2-DFA.png)

(b)

```
Sinit:
	lexeme = "";
	list = {"abc", "bca", "cab"}
	goto S0;
	
S0:
	NextChar(char);
	lexeme = lexeme + char;
	if ('a' <= char <= 'c')
     	goto S1;
   	else
   		return invalid;

S1:
	NextChar(char);
	lexeme = lexeme + char;
	if ('a' <= char <= 'c')
     	goto S2;
   	else
   		return invalid;
   		
S2:
	NextChar(char);
	lexeme = lexeme + char;
	if ('a' <= char <= 'c')
     	goto S3;
   	else
   		return invalid;

S3:
	if (lexeme in list)
		return valid;
	else
		return invalid;
```

(c) 假定一个状态的代码片段的开销为n，查表的开销为m，那么

- 本方法的开销为3 * n + m
